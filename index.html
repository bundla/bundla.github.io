BUNDLA

A console utility for raising the development environment on Laravel + Docker.

Installing Bundla

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/tgaru/bundla/master/install.sh)"

Paste that in a macOS Terminal or Linux shell prompt.

The script explains what it will do and asks for confirmation to continue.
